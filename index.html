Title of Project: Speech Emotion Recognition
Advisor: Nurdan Saran

Project Description

This system makes an emotion analysis from voice data and produces a result among the options "happy", "angry", "sad", "neutral". It is planned to be used both in a system that normal people can use in their daily lives and in a system where priority is given to the emotional state of the customer, especially for those working in the call center.
